## Cursor selectors

In order to properly style your cursor, here are the selectors you must use :

```css
/* Cursor shape */
.cursor::before {
    ...
}

/* Cursor when the user clicks anywhere */
.cursor.--active::before {
    ...
}

/* Cursor when visible */
.cursor.--visible::before {
    ...
}

/* Cursor when hidden */
.cursor.--hidden::before {
    ...
}

/* Cursor text */
.cursor__text {
    ...
}
```
